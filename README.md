
# Running the Project

## Create an env file with your Notion credentials

Create a `server/.env` file:

```
NOTION_SECRET=
NOTION_DATABASE_ID=
```

## Run the server

From the `/server` directory:

```
npm run start
```

## Run the app

From the `/sample-app` directory:

```
npm run start
```

Connect on [http://localhost:3000/]()
